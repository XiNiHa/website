병원에서 환자 예약부터 진료, 혈액검사 예약, 문진 작성, 리포트 제공, 투약 알림까지<br>
전 과정에 대한 의사 대상 UI와 환자 대상 모바일 웹앱을 작업하였습니다.

원활한 SSR을 위해 Prefetch 등의 최적화를 진행하였습니다.

또한 작업 중 API 백엔드인 Falcon 기반 Python 코드의 유지보수 및 기능 추가도 같이 진행하였습니다.
